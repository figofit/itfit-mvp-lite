# ITFIT MVP Lite

Jednoduchý MVP dashboard postavený na Next.js + TypeScript + Tailwind.

## Spuštění

```bash
npm install
npm run dev
```

## Ukládání dat

Data se ukládají do LocalStorage pod klíčem `itfit:v1`.

## Zálohy

- Export JSON i CSV najdeš v sekci **Nastavení → Zálohy**.
- Import JSON nejdřív ověří strukturu a pak nabídne přepsání dat.
